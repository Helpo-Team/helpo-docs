# Docs

[Documentacion](https://helpo-docs.readthedocs.io/en/latest/?)

## Instalar 

`pip install -U sphinx`

`pip install sphinx_rtd_theme`

## Buildear

(Buildear documentacion para poder ver los cambios)

Linux:

`make html`

Windows:

`make.bat html`

## Ver Doc

Abrir con navegador helpo-docs/_build/html/index.html

## NOTA

Cada vez que se pushea al repositorio el servidor que muestra nuestra documentacion la buildea de nuevo.
